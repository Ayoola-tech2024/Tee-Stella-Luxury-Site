# Tee Stella's Fabrics & Stitches - Interaction Design

## Core Interactive Components

### 1. Product Filter System (Shop Page)
- **Functionality**: Filter products by category (Women, Men, Ready-to-Wear, Accessories)
- **Interaction**: Click filter buttons to show/hide products instantly
- **Visual Feedback**: Active filter button highlighted in gold, smooth transitions
- **Default State**: Show all products on page load

### 2. Product Gallery Carousel
- **Functionality**: Each product displays 1-6 images in a horizontal carousel
- **Interaction**: Click navigation arrows or swipe on mobile to browse images
- **Visual Feedback**: Smooth slide transitions, dot indicators for current image
- **Auto-play**: Optional 5-second interval auto-rotation

### 3. WhatsApp Integration System
- **Product Order Button**: Opens WhatsApp with pre-filled message including product details
- **Bespoke Order Button**: Direct WhatsApp link for custom orders
- **Floating WhatsApp Button**: Persistent button visible on all pages
- **Message Format**: "Hello Tee Stella's, I'm interested in this product. Product: [NAME] Price: [PRICE] Picture: [IMAGE_URL] Please confirm availability."

### 4. Testimonials Slider (Home Page)
- **Functionality**: Automatic sliding testimonials with manual navigation
- **Interaction**: Auto-advance every 4 seconds, manual next/prev buttons
- **Visual Feedback**: Smooth fade transitions, progress indicator
- **Content**: Minimum 3 testimonials with customer photos

### 5. Instagram Gallery Integration
- **Functionality**: Display latest Instagram posts in a grid
- **Interaction**: Click images to open in new tab to Instagram
- **Visual Feedback**: Hover effects with subtle zoom and overlay
- **Layout**: Responsive grid adapting to screen size

### 6. Contact Form (About & Contact Page)
- **Functionality**: Name, Email, Message fields with validation
- **Interaction**: Real-time validation, success/error states
- **Submission**: Sends email to olutadehormololar@gmail.com
- **Visual Feedback**: Loading states, success confirmation, error handling

## User Journey Flow

### Shopping Experience
1. User lands on homepage → sees hero banner and featured products
2. Clicks "Shop Now" → navigates to shop page
3. Uses filters to find desired category → products filter instantly
4. Clicks product → views image gallery
5. Clicks "Order via WhatsApp" → opens WhatsApp with pre-filled message
6. Completes order conversation in WhatsApp

### Bespoke Experience
1. User navigates to Bespoke Orders page
2. Reads about custom tailoring services
3. Clicks "Start Custom Order on WhatsApp" → direct WhatsApp link
4. Begins consultation conversation immediately

### Contact Experience
1. User visits About & Contact page
2. Reads brand story and mission
3. Fills contact form → receives confirmation
4. Can also click social media icons or call directly

## Mobile Responsiveness
- Touch-friendly button sizes (minimum 44px)
- Swipe gestures for carousels
- Collapsible navigation menu
- Optimized WhatsApp integration for mobile
- Responsive image galleries

## Accessibility Features
- Alt text for all images
- Keyboard navigation support
- Screen reader friendly
- High contrast text (4.5:1 ratio minimum)
- Focus indicators for interactive elements
